# スキル健全性ログ

スキルの実行品質を記録し、改善につなげるファイル。

## 自動記録ルール

### いつ記録するか
- スキル実行後、期待通りの結果が得られなかった場合
- スキル実行後、手動で修正を加えた場合

### いつ改善するか
- 同じスキルに2件以上の問題が記録されていたら、SKILL.mdの改善を検討する

## 記録フォーマット

### [スキル名] 問題の簡潔な説明
- **発生日**: YYYY-MM-DD
- **案件**: プロジェクト名
- **症状**: 何が期待と違ったか
- **原因推定**: モデル変化 / 環境変化 / トリガー不一致 / 指示の曖昧さ
- **ステータス**: 未対応 / 修正済み / 解消確認済み
